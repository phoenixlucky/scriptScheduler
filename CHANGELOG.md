# Changelog

## 1.1.9 - 2026-03-23

- 优化 Conda 环境解析：优先读取 `conda env list --json` / `conda info --envs --json`，按环境名定位真实路径，减少对固定盘符和用户名的依赖。
- 为 `conda-name` / `conda-path` 增加执行回退策略：找不到 `python.exe` 时自动回退到 `conda run --no-capture-output`。
- 改进手动执行失败提示：点击“立即执行”或“立即执行1次”失败时，弹窗显示更具体的错误内容。
- 改进任务状态展示：失败任务在列表中显示更明确的失败原因。
- 新增 Cron 五段自由组合输入，并与原始 Cron 表达式、预设按钮双向同步。
- 修复失败日志可能被覆盖、导致只看到 `failed` 看不到具体错误的问题。
